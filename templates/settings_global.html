Global Settings

Global settings define system-wide behaviour that applies across all grow profiles. Adjust these values to control safety thresholds, hysteresis buffers, and hardware behaviour such as the agitator pre-mix. These apply at all times, regardless of which profile is running.

Hysteresis
Set per-device deadbands here. These values take priority. If a device value isn’t set, the global fallbacks below will be used instead.
Fallback hysteresis (legacy/global): These are only used when a device-specific value above is not set. Prefer setting the per-device fields.
Agitator (pre-mix)
Example: run 15 s → agitator pump turns on 15 s before irrigation and stops when the main irrigation starts.
Air Pump (pre-mix)

Absolute Safety Limits
If readings go outside these, controls are cut and an alert is raised.
Water Temperature
Used for display/banners, does not control shutdown.
Reservoir
Gross weight of the reservoir hardware with no water. Used to subtract and calculate net water weight.
Net water at your “full” line (no hardware included). The app derives gross full = empty gross + this capacity.
Water mass at which the UI will display the status “Half”. Usually ~50% of usable capacity.
Threshold for “Low” status. Indicates water is running down but still above critical/cutoff.
Threshold for “Critical” status. Warning only — does not stop the pump. Set higher than cutoff to give early notice.
Safety shutdown threshold. If water falls below this, the pump is shutdown, all outputs are stopped, and an alert is raised via Discord.
Margin for showing “Full”. Example: if usable = 20 kg and margin = 1 kg, the UI shows “Full” down to 19 kg to avoid flicker.
All thresholds are in kg of water (gross weight − empty weight). “Critical” gives a warning; “Cutoff” actually stops pump cycles and raises an alert.
